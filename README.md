# AgriConnect 🌱

## Project Overview

AgriConnect is a web application designed to connect farmers and consumers directly. It helps users access agricultural products, services, and information efficiently.

## Features

* User-friendly interface
* Product listing and browsing
* Backend integration using Spring Boot
* Real-time updates

## Technologies Used

* React (Frontend)
* TypeScript
* Tailwind CSS
* Spring Boot (Backend)
* MySQL (Database)

## Installation & Setup

### Clone the repository

```bash
git clone <your-repo-url>
cd <project-folder>
```

### Frontend Setup

```bash
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Database Setup

* Create a MySQL database named `agriconnect`
* Update `application.properties` with your credentials
